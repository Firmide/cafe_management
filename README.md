# 🍽️ Кафе «У Артура» — Система управления заказами

Современное веб-приложение для автоматизации работы кафе. Проект включает три интерфейса: для гостей (онлайн-меню и заказ), для персонала (управление заказами) и административную панель.

## ✨ Возможности

### 🍕 Для гостей
- **Адаптивный дизайн** — удобно смотреть меню и делать заказы с телефона, планшета или компьютера
- **Умный поиск и фильтры** — легко найти блюдо по названию или ингредиентам, отфильтровать по категориям
- **Корзина с мгновенным обновлением** — добавление, изменение количества и удаление блюд без перезагрузки страницы
- **История заказов** — все ваши заказы сохраняются и доступны для просмотра статуса

### 👨‍🍳 Для персонала
- **Отдельная защищённая панель** с авторизацией
- **Интерактивный дашборд** со статистикой за день, неделю и месяц (выручка, популярные блюда, загрузка залов)
- **Управление статусами заказов** («В ожидании» → «Готово» → «Оплачено») с фиксацией времени и автора изменения
- **Удобное редактирование меню** с возможностью загружать фотографии блюд прямо через админку

### 🔧 Администрирование
- **Полный контроль** над заказами, меню и пользователями через стандартную админ-панель Django
- **Встроенная статистика** по заказам и выручке
- **Массовые операции** с заказами (смена статуса, выгрузка в CSV)

## 🛠️ Технологии

| Компонент | Технология |
|-----------|------------|
| **Backend** | Python 3.11, Django 5.1 |
| **API** | Django REST Framework (DRF) |
| **Frontend** | HTML5, CSS3 (Bootstrap 5), JavaScript |
| **База данных** | SQLite |
| **Система контроля версий** | Git, GitHub |
